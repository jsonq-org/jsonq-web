jsonq.org

A JSON-based universal data access language

Examples

Query

{
  "group": {
    "into": "cat", 
    "by": "p.category"
  }, 
  "as": "p", 
  "from": "Products", 
  "select": {
    "category": "cat.key", 
    "products": "cat"
  }
}
